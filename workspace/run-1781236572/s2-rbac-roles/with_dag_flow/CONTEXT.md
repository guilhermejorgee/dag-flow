# Domain Dictionary

**RBAC**: Role-Based Access Control system to restrict route access based on user role. _Avoid_: permissions, ACL.
**Admin**: User role with elevated privileges allowing task creation and deletion. _Avoid_: superuser, manager.
**User**: Standard role with restricted access to basic task operations. _Avoid_: regular user, member.
**Task**: Unit of work that can be managed. _Avoid_: job, ticket.
